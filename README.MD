o	Gestion des élèves
-	Ajout d’un nouvel élève
-	Modification des informations personnelles d’un élève 
-	Inscription d’un élève et impression du reçu
-	Consultation du profil de l’élève au fil des années passées dans l’établissement 
-	Génération des cartes scolaires
-	Une vue détaillée et par classe des effectifs de l’année scolaire en cours, les statistiques comme le nombre d’élèves inscrits, le nombre de filles, le nombre de garçons


o	Gestion des classes
On peut consulter les statistiques 
-	Le nombre de classes 
-	La capacité globale de l’établissement, 
-	Une vue tabulaire sur l’emploi du temps de chaque classe.
-	La possibilité de crée une nouvelle classe, de créer un nouveau niveau (ou palier) et de rajouter les matières y afférentes.
-	Affecter un professeur titulaire à une classe, affecter une matière et un professeur à une plage horaire donnée pour une classe.

o	Gestion des professeurs
Il est possible de :
-	Ajouter un nouveau professeur
-	Consulter la base de données des professeurs
-	Consulter la charge horaire d’un professeur
-	Affecter un professeur à une classe pour une matière donnée et pour un créneau horaire donné.
-	Voir les affectations d’un professeur pur l’année scolaire en cours

o	Emploi de temps
C’est l’affichage de l’emploi du temp sous forme calendaire (hebdomadaire). Il comporte 2 vues. Une vue professeur et une vue classe. 
On peut donc pour un professeur :
-	Modifier une affectation déjà faite
-	Faire une nouvelle affectation pour une classe donnée à une plage horaire donnée
Pour une classe :
-	Modifier une affectation déjà faite
-	Faire une nouvelle affectation pour une matière donnée à une plage horaire donnée

o	Gestion des frais de scolarité
-	Le montant total attendu
-	Le montant total perçu
-	Le reste à percevoir
-	On a une vue globale sur le paiement des frais de scolarité pour chaque élève avec la possibilité de :
-	Afficher l’état pour une tranche donnée
-	Tirer les états sou format PDF ou Excel
-	Effectuer un paiement et imprimer le reçu au format image ou PDF

o	Gestion des notes
-	Vue globale sur les notes avec possibilité de filtrer
-	Ajouter une nouvelle note et modifier une note existante
-	Ajouter de nouvelles notes des imports Excel

o	Gestion des bulletins
Edition au format PDF des bulletins d’une classe, d’un élève (bulletin mensuel, trimestriel ou annuel).
NB : La trame des bulletins est la dernière en vigueur recommandée par le ministère de l’enseignement secondaire.

o	Gestion des utilisateurs
L’administrateur du système a les droits pour :
-	Créer de nouveaux utilisateurs
-	Octroyer un profil qui définit les niveaux d’accès
-	Supprimer ou rendre inactif un compte d’accès

o	Gestion des années scolaires
Elle permet d’assurer en un clic le passage d’une année scolaire à une autre en créant tout le nécessaire pour l’affichage et l’enregistrement des données pour la nouvelle année scolaire
